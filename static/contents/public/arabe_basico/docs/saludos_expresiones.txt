SALUDOS Y EXPRESIONES BÁSICAS EN ÁRABE
========================================

Una recopilación de las frases más útiles para empezar a comunicarse
en árabe estándar moderno (فُصْحَى, fuṣḥā).


1. SALUDOS

   السَّلَامُ عَلَيْكُمْ   (as-salāmu ʿalaykum)
   → La paz sea con vosotros.
   Respuesta: وَعَلَيْكُمُ السَّلَام (wa ʿalaykumu s-salām)

   مَرْحَبًا   (marḥaban)
   → Hola. Saludo informal y universal.

   صَبَاحُ الخَيْر   (ṣabāḥu l-jayr)
   → Buenos días. (lit. "Mañana de bien")
   Respuesta: صَبَاحُ النُّور (ṣabāḥu n-nūr) — "Mañana de luz"

   مَسَاءُ الخَيْر   (masāʾu l-jayr)
   → Buenas tardes / noches.
   Respuesta: مَسَاءُ النُّور (masāʾu n-nūr)


2. PRESENTARSE

   مَا اسْمُكَ؟   (mā ismuka? / mā ismuki?)
   → ¿Cómo te llamas? (-ka para hombre, -ki para mujer)

   اِسْمِي...   (ismī...)
   → Mi nombre es...

   أَنَا مِنْ...   (anā min...)
   → Soy de...

   تَشَرَّفْنَا   (tasharrafnā)
   → Encantado/a de conocerte.


3. EXPRESIONES COTIDIANAS

   شُكْرًا   (shukran)
   → Gracias.

   عَفْوًا   (ʿafwan)
   → De nada. / Disculpe.

   مِنْ فَضْلِكَ   (min faḍlika / min faḍliki)
   → Por favor.

   نَعَمْ   (naʿam)
   → Sí.

   لَا   (lā)
   → No.

   مَعَ السَّلَامَة   (maʿa s-salāma)
   → Adiós. (lit. "Con la paz/seguridad")


4. PREGUNTAS ÚTILES

   كَيْفَ حَالُكَ؟   (kayfa ḥāluka? / ḥāluki?)
   → ¿Cómo estás?
   Respuesta: بِخَيْر، الحَمْدُ لِلَّه (bi-jayr, al-ḥamdu li-llāh)
   → Bien, gracias a Dios.

   أَيْنَ...؟   (ayna...?)
   → ¿Dónde está...?

   كَمْ؟   (kam?)
   → ¿Cuánto? / ¿Cuántos?

   لِمَاذَا؟   (limāḏā?)
   → ¿Por qué?

   هَلْ تَتَكَلَّمُ الإِسْبَانِيَّة؟   (hal tatakallamu l-isbāniyya?)
   → ¿Hablas español?


5. FRASES DEL DÍA A DÍA

   إِنْ شَاءَ اللَّه   (in shāʾa llāh)
   → Si Dios quiere. (Se usa constantemente para planes futuros)

   مَا شَاءَ اللَّه   (mā shāʾa llāh)
   → Lo que Dios ha querido. (Admiración, para evitar el mal de ojo)

   الحَمْدُ لِلَّه   (al-ḥamdu li-llāh)
   → Gracias a Dios. (Se usa al responder "¿cómo estás?", al terminar de comer, etc.)

   بِسْمِ اللَّه   (bismi llāh)
   → En el nombre de Dios. (Se dice antes de comer, empezar algo, etc.)
